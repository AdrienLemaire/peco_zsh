# peco zsh

zsh plugin for the [peco](https://github.com/peco/peco) tool

## Install

with [antigen](https://github.com/zsh-users/antigen), add in your `~/.zshrc`:

    antigen bundle Fandekasp/peco_zsh

## Features

| trigger   | action                  |
| --------- | ----------------------- |
| s\<Enter> | Lists all ssh hostnames |
